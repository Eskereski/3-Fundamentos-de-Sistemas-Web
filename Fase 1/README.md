# Petshop Web - Fase 1

## Descrição
Sistema web estático para um petshop, desenvolvido como primeira fase do projeto de Fundamentos de Sistemas Web. A implementação utiliza exclusivamente HTML5 para a marcação semântica, sem a aplicação de CSS ou JavaScript nesta etapa.

## Funcionalidades
- **Catálogo de Produtos:** Exibição de itens com imagem, descrição e valor, organizados nas categorias:
  - Acessórios
  - Rações não perecíveis
  - Produtos de higiene e limpeza
- **Serviços:** Listagem de serviços do petshop (banho e tosa), incluindo descrição, valor e disponibilidade de tele-busca.

## Tecnologias
- HTML5
